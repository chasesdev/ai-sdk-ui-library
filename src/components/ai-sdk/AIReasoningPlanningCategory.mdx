# AI Reasoning & Planning Components

Components for displaying AI reasoning processes, thought chains, tool usage, plans, and task execution. These components help users understand how AI systems think, make decisions, and execute multi-step workflows.

---

## Component Index

- [Reasoning](#reasoning) - Display AI reasoning steps
- [ChainOfThought](#chainofthought) - Visualize thought processes
- [Tool](#tool) - Display tool/function calls
- [Response](#response) - Formatted AI responses
- [Context](#context) - Contextual information display
- [Plan](#plan) - AI-generated plans
- [Task](#task) - Individual task tracking

## Usage Together

These components work together to create transparent AI workflows:

```tsx
import { Reasoning, ChainOfThought, Tool, Plan, Task } from '@/components/ai-sdk';

function AIWorkflow() {
  return (
    <div>
      <Reasoning steps={reasoningSteps} />

      <ChainOfThought thoughts={thoughts} />

      <Tool
        name="web_search"
        input={{ query: "latest AI news" }}
        output={searchResults}
        status="completed"
      />

      <Plan steps={planSteps} />

      {tasks.map(task => (
        <Task key={task.id} {...task} />
      ))}
    </div>
  );
}
```

## Best Practices

1. **Transparency**: Show reasoning steps to build user trust
2. **Progressive Disclosure**: Use collapsible sections for detailed information
3. **Status Indicators**: Clearly show execution status (pending, running, completed, failed)
4. **Error Handling**: Display meaningful error messages when tools or tasks fail
5. **Performance**: Lazy-load detailed reasoning data for large workflows
