## About

These components are built with modern web technologies and best practices:

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type-safe development
- **Tailwind CSS** - Utility-first styling
- **shadcn/ui** - Beautiful, accessible components
- **Radix UI** - Unstyled, accessible primitives

All components support **dark mode** out of the box using Tailwind's class-based dark mode. Use the background selector (paint palette icon) in the Storybook toolbar above to switch between light and dark themes!

## Features

### Dark Mode Support

All components are designed with dark mode in mind. They use semantic color tokens that automatically adapt to the selected theme:

- `bg-background` / `bg-foreground`
- `text-foreground` / `text-muted-foreground`
- `border-border`
- `bg-primary` / `text-primary`

**To toggle dark mode in Storybook:** Click the background color selector in the toolbar at the top (looks like a paint palette icon) and switch between "light" and "dark" backgrounds. The components will automatically update to show their dark mode variants!

### Accessibility

Components are built with accessibility in mind using:

- Semantic HTML elements
- ARIA attributes where needed
- Keyboard navigation support
- Screen reader friendly

### Responsive Design

Components are responsive and work across different screen sizes:

- Mobile (320px-767px)
- Tablet (768px-1199px)
- Desktop (1200px+)

Check out our [showcase pages](/) to see components at different breakpoints!

### TypeScript Support

All components are written in TypeScript with full type definitions for props and return types.
