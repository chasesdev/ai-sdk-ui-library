## Usage Together

Combine these components to create rich action interfaces:

```tsx
import { Actions, Controls, Toolbar } from '@/components/ai-sdk';

function ActionInterface() {
  return (
    <div>
      <Toolbar>
        <Actions
          actions={[
            { label: 'Save', icon: 'save', onClick: handleSave },
            { label: 'Export', icon: 'download', onClick: handleExport },
          ]}
        />
      </Toolbar>

      <Controls
        onPlay={handlePlay}
        onPause={handlePause}
        onStop={handleStop}
        onSkip={handleSkip}
      />
    </div>
  );
}
```

## Best Practices

1. **Icon Clarity**: Use universally recognized icons (play, pause, stop, etc.)
2. **Keyboard Shortcuts**: Implement keyboard shortcuts for common actions
3. **Disabled States**: Clearly indicate when actions are unavailable
4. **Tooltips**: Provide helpful tooltips for all action buttons
5. **Responsive Layout**: Collapse or reorganize actions on smaller screens
6. **Loading States**: Show progress indicators during async actions
