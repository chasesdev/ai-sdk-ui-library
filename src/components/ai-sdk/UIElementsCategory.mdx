## Usage Together

Combine UI elements to create polished, professional interfaces:

```tsx
import { Loader, Shimmer, InlineCitation, Sources, OpenInChat } from '@/components/ai-sdk';

function AIResponse({ loading, content, sources }) {
  if (loading) {
    return (
      <div>
        <Shimmer type="text" lines={3} />
        <Shimmer type="card" />
      </div>
    );
  }

  return (
    <div>
      <p>
        {content}
        <InlineCitation number={1} source={sources[0]} />
      </p>

      <Sources sources={sources} />

      <OpenInChat
        text="Continue in chat"
        onClick={handleOpenChat}
      />
    </div>
  );
}
```

## Best Practices

1. **Loading States**: Always show loading indicators for async operations
2. **Skeleton Screens**: Use Shimmer for better perceived performance
3. **Citation Accuracy**: Ensure citations match actual sources
4. **Source Verification**: Link to original sources when possible
5. **Consistent Styling**: Maintain consistent visual language across UI elements
6. **Accessibility**: Provide aria-labels for loading states and interactive elements
7. **Performance**: Lazy-load sources and citation details
