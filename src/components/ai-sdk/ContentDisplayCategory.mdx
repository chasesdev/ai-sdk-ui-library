# Content Display Components

Components for displaying various types of AI-generated content including code, artifacts, images, web previews, and structured panels. These components ensure content is presented in an accessible, visually appealing format.

---

## Component Index

- [Artifact](#artifact) - Display generated artifacts
- [BracesCodeBlock](#bracescodeblock) - Syntax-highlighted code blocks
- [Canvas](#canvas) - Drawing and visualization canvas
- [Image](#image) - AI-generated or referenced images
- [WebPreview](#webpreview) - Preview web pages and links
- [Panel](#panel) - Container panels with controls

## Usage Together

Combine content display components for rich multimedia experiences:

```tsx
import { Artifact, BracesCodeBlock, Image, WebPreview, Panel } from '@/components/ai-sdk';

function ContentDisplay() {
  return (
    <Panel title="AI Generated Content">
      <Artifact type="code">
        <BracesCodeBlock
          language="typescript"
          code={generatedCode}
        />
      </Artifact>

      <Image
        src={generatedImage}
        caption="AI-generated visualization"
      />

      <WebPreview
        url="https://example.com"
        title="Related Resource"
      />
    </Panel>
  );
}
```

## Best Practices

1. **Lazy Loading**: Load heavy content (images, previews) on-demand
2. **Copy Actions**: Provide easy copy functionality for code and text
3. **Syntax Highlighting**: Use appropriate language-specific highlighting
4. **Error States**: Gracefully handle missing or failed content
5. **Responsive Design**: Ensure content adapts to different screen sizes
6. **Accessibility**: Include alt text for images and semantic HTML
